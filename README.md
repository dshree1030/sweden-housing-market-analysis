# Sweden Housing Market Analysis (2024 - 2026)

## 📌 Project Overview
This project investigates the stabilization and recovery trends of the Swedish property market following the 2023 economic correction. Using **SQL**, I analyzed regional price fluctuations with a specific focus on the **Gothenburg (Västra Götaland)** area.

## 🛠️ Technical Stack
* **Language:** SQL (PostgreSQL / MySQL)
* **Key Techniques:** Data Cleaning, Window Functions (RANK, LEAD/LAG), CTEs, and Joins.
* **Data Sources:** Statistics Sweden (SCB) & Kaggle Housing Datasets.

## ❓ Business Questions Answered
1. Which municipalities in the Gothenburg region showed the fastest price recovery?
2. What is the average price-per-square-meter gap between Stockholm and Gothenburg?
3. How did interest rate shifts impact transaction volumes in Q1 2026?

## 📊 Sample Insights (Work in Progress)
* **Volatility:** Apartment prices in central Gothenburg stabilized 4 months faster than standalone villas.
* **Growth Zones:** Mölndal and Partille emerged as high-demand zones with a 5% increase in YoY volume.
